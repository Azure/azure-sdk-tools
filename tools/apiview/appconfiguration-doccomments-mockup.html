ConfigurationAsyncClient

public final class ConfigurationAsyncClient

Provides an asynchronous client containing operations for ConfigurationSetting, FeatureFlagConfigurationSetting, and SecretReferenceConfigurationSetting resources in an Azure App Configuration store.

Operations include adding, retrieving, deleting, setting read-only status, listing settings, listing revisions, and adding external synchronization tokens with updateSyncToken(String).

ConfigurationAsyncClient configurationAsyncClient = new ConfigurationClientBuilder()
    .connectionString(connectionString)
    .buildAsyncClient();
